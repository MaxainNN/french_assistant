# Артикли и существительные во французском языке

## Определённые артикли (Articles définis)

| Род/Число | Артикль | Пример |
|-----------|---------|--------|
| Мужской род ед.ч. | le | le livre (книга) |
| Женский род ед.ч. | la | la table (стол) |
| Перед гласной | l' | l'école (школа), l'homme (мужчина) |
| Множ. число | les | les livres, les tables |

**Употребление:**
- Конкретный, известный предмет: Le livre est sur la table. (Книга на столе.)
- Обобщение: Le vin français est célèbre. (Французское вино знаменито.)
- С названиями стран: la France, le Canada, les États-Unis
- С абстрактными понятиями: l'amour, la liberté, le bonheur

## Неопределённые артикли (Articles indéfinis)

| Род/Число | Артикль | Пример |
|-----------|---------|--------|
| Мужской род | un | un livre (книга) |
| Женский род | une | une table (стол) |
| Множ. число | des | des livres, des tables |

**Употребление:**
- Неизвестный/впервые упоминаемый предмет: J'ai acheté un livre. (Я купил книгу.)
- Один из многих: C'est une bonne idée. (Это хорошая идея.)

**ВАЖНО:** После отрицания des → de/d':
- J'ai des amis. → Je n'ai pas d'amis. (У меня нет друзей.)
- Elle a des livres. → Elle n'a pas de livres. (У неё нет книг.)

## Частичные артикли (Articles partitifs)

| Род | Артикль | Пример |
|-----|---------|--------|
| Мужской род | du | du pain (хлеб), du café (кофе) |
| Женский род | de la | de la viande (мясо), de la musique (музыка) |
| Перед гласной | de l' | de l'eau (вода), de l'argent (деньги) |
| Множ. число | des | des épinards (шпинат) |

**Употребление:**
- Неопределённое количество: Je veux du café. (Я хочу кофе.)
- Неисчисляемые существительные: Il y a de la neige. (Есть снег.)

**ВАЖНО:** После отрицания du/de la/de l' → de/d':
- Je bois du lait. → Je ne bois pas de lait. (Я не пью молоко.)

## Слитные артикли (Articles contractés)

| Предлог + артикль | Слитная форма |
|-------------------|---------------|
| à + le | au (au cinéma) |
| à + les | aux (aux étudiants) |
| de + le | du (du magasin) |
| de + les | des (des enfants) |

**Примеры:**
- Je vais au (à + le) bureau. (Я иду в офис.)
- Je parle aux (à + les) étudiants. (Я говорю со студентами.)
- Je viens du (de + le) restaurant. (Я иду из ресторана.)
- C'est la maison des (de + les) voisins. (Это дом соседей.)

**ВАЖНО:** la и l' не сливаются:
- à la maison, à l'école
- de la maison, de l'école

## Род существительных

### Мужской род (masculin)
**Окончания обычно мужского рода:**
- -age: le voyage, le message (искл.: la page, la plage, l'image)
- -ment: le moment, le gouvernement
- -eau: le bureau, le château
- -isme: le tourisme, le capitalisme
- -eur: le professeur, le facteur (но: la peur, la fleur, la douleur)

**Всегда мужской род:**
- Дни недели: le lundi, le mardi
- Месяцы: le janvier, le février
- Времена года: le printemps, l'été, l'automne, l'hiver
- Языки: le français, le russe, l'anglais

### Женский род (féminin)
**Окончания обычно женского рода:**
- -tion, -sion: la nation, la télévision
- -té: la liberté, la beauté
- -ure: la nature, la culture
- -ette: la cigarette, la fourchette
- -ence, -ance: la différence, la distance
- -ie: la vie, la philosophie

**Всегда женский род:**
- Науки: la chimie, la physique, la biologie
- Автомобильные марки: une Renault, une Peugeot

## Образование множественного числа

### Общее правило
Добавляется -s: livre → livres, table → tables

### Особые случаи
| Окончание ед.ч. | Множ. число | Пример |
|-----------------|-------------|--------|
| -s, -x, -z | без изменений | le bus → les bus, le nez → les nez |
| -au, -eau, -eu | + x | le bureau → les bureaux, le jeu → les jeux |
| -al | -aux | le journal → les journaux, l'animal → les animaux |
| -ail | -aux (некоторые) | le travail → les travaux, le vitrail → les vitraux |

### Исключения
- l'œil → les yeux (глаз → глаза)
- le ciel → les cieux (небо → небеса)
- madame → mesdames
- monsieur → messieurs
- mademoiselle → mesdemoiselles

## Образование женского рода от мужского

### Общее правило
Добавляется -e: ami → amie, étudiant → étudiante

### Особые случаи
| Мужской род | Женский род | Пример |
|-------------|-------------|--------|
| -er | -ère | boulanger → boulangère |
| -eur | -euse | vendeur → vendeuse |
| -teur | -trice | acteur → actrice |
| -f | -ve | sportif → sportive |
| -x | -se | heureux → heureuse |
| -en, -on | удвоение + e | musicien → musicienne |

### Разные формы
- homme → femme
- garçon → fille
- père → mère
- frère → sœur
- oncle → tante
- neveu → nièce
- mari → femme
- roi → reine
